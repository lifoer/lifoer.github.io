9.linux删除文本中^M
1.^M引起原因：“\r”回车符
2.查看^M

```
cat -A file.txt
```

3.删除^M，^M不可复制，手动按出：ctrl+v+m

```
cat file.txt | tr -d "^M" > newfile.txt
```

4.批量处理
* 1). 批量删除

```
#!/bin/bash
for file in `ls | grep .txt`
do
    newfile=`echo $file | sed "s/\([a-zA-Z0-9]\+\)\(.txt\)/\1-f\2/"`
    cat $file | tr -d "^M" > $newfile
done
```

* 2).批量验证

```
#!/bin/bash
for file in `ls | grep .txt`
do 
    printf "\n%s\n" $file
    cat -A $file
done
```
